# SideDeck

SideDeck is an open-source Android launcher created for handheld gaming devices and controller-first setups. It provides a streamlined, console-like interface that prioritizes quick access to games and essential apps.

## Features
• Full controller support and key mapping  
• Game-focused home screen and library view  
• Performance focused with minimal resource usage  
• Landscape optimized for handheld displays  
• Customizable UI elements and layout behavior  
• Unified access to installed and sideloaded apps  
• Clean and distraction-free design

SideDeck gives you a gaming interface without removing the flexibility of Android. It is ideal for handheld consoles, gaming phones, and docked living room setups.

## Status

SideDeck is currently in active development. Features may change as the project evolves. Testing builds and releases will be published here when available.
