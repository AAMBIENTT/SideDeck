# SideDeck

SideDeck is an open-source Android launcher created for handheld gaming devices and controller-first setups. It provides a streamlined, console-like interface that prioritizes quick access to games and essential apps.

## Features:
• Full controller support and Keymapping

• Game-focused home screen and library view

• Performance focused with minimal resource usage

• Landscape optimized for handheld displays

• Customizable UI elements and layout behavior

SideDeck gives you a gaming interface without removing the flexibility of Android.
